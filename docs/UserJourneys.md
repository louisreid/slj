# User Journeys (V1)

## First-time user
1) Open course site
2) Enter email → receive magic link
3) Sign in
4) Reader opens at beginning (or a “Resume” prompt if progress exists)
5) Create a note next to a paragraph
6) Mark section complete

Error states:
- invalid/expired magic link → request new link
- offline → show “can’t save note” and retry

## Returning user
1) Sign in
2) Resume at last location
3) View notes for current chapter

## Print worksheet
1) Open Worksheets list
2) Choose worksheet
3) Click Print
4) Print dialog opens in a clean print-only layout

## Join group
1) Open Groups
2) Enter invite code / open invite link
3) See start date and shared group notes block
4) Edit shared group notes block (autosave)

Constraints:
- group notes block is shared
- course notes remain private and never appear in groups
